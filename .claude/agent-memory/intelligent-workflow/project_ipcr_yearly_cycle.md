name: IPCR Yearly Cycle
description: The four-step IPCR yearly cycle and which months each step occurs
type: project
---

The IPCR follows a strict yearly cycle:

1. **First Semester Targets** (submitted in November): Employees set targets for Jan–June of the NEXT year.
2. **First Semester IPCR Submission** (submitted in July): Employees submit actual accomplishments for Jan–June. References the November targets.
3. **Second Semester Targets** (submitted in May): Employees set targets for Jul–December of the CURRENT year.
4. **Second Semester IPCR Submission** (submitted in January): Employees submit actual accomplishments for Jul–Dec. References the May targets.

**Why:** This is the CSC-mandated IPCR cycle for Philippine government employees.

**How to apply:** When writing routing logic, period calculations, or target-to-submission linkage, always keep this cycle in mind. Semester 1 target_year = next year when submitted in November. Semester 2 target_year = current year when submitted in May.
